**Constants:**
- QDRANT_URL = "https://f969d77f-3cf6-4557-92cb-67f7cac0f44a.us-west-1-0.aws.cloud.qdrant.io:6333"
- COLLECTION = "knowledge_docs"
- EMBED_MODEL = "models/gemini-embedding-2-preview"
- DIMS = 3072
- CHUNK_SIZE = 1000 (characters)
- CHUNK_OVERLAP = 100

**Safety guard (CRITICAL):**
- `assert COLLECTION != "agent_memory", "NEVER touch agent_memory collection"`
- This prevents accidental corruption of mem0's vector store

**Imports:**
- `os, hashlib, time, sys` from stdlib
- `pathlib.Path`
- `qdrant_client.QdrantClient`
- `qdrant_client.models.VectorParams, Distance, PointStruct`
- `google.genai` (the google-genai 1.67.0 SDK)

**PATHS to index (5 directories):**
```python
PATHS = [
    Path.home() / "github/agent-lightning",
    Path.home() / "github/agent-lightning/docs",
    Path.home() / "github/knowledge/llm-wiki/wiki",
    Path.home() / "github/knowledge",
    Path.home() / "github/knowledge/mem0-exports",
]
```

**Functions:**
1. `get_client()` — returns QdrantClient(url=QDRANT_URL, api_key=os.environ["QDRANT_API_KEY"])
2. `ensure_collection(client)` — try get_collection, except create_collection with VectorParams(size=DIMS, distance=Distance.COSINE). Print status either way.
3. `chunk_text(text, size, overlap)` — sliding window chunker, returns list of strings
4. `embed(genai_client, text)` — calls genai_client.models.embed_content(model=EMBED_MODEL, contents=text), returns resp.embeddings[0].values
5. `stable_id(filepath, chunk_idx)` — SHA256 hash of "{filepath}:{chunk_idx}", return first 16 hex chars as int (64-bit deterministic ID for idempotent upsert)
6. `index_paths(client, genai_client, paths)` — for each base path, rglob("*.md"), read text, chunk, embed each chunk with 0.1s sleep between embed calls (rate limit protection), build PointStruct list with payload {file, collection, chunk_index, text[:500]}, upsert batch per file. Print progress per file.
   - Skip empty chunks (whitespace only)
   - Skip files that fail to read (errors="ignore" on read_text, try/except around embed)
   - Add retry logic: if embed call fails, sleep 2s and retry once before skipping chunk
   - Exclude paths containing `node_modules`, `.git`, `.venv`, `__pycache__` directories

**Main block:**
- genai_client = genai.Client() (reads GEMINI_API_KEY from env automatically)
- qdrant_client = get_client()
- ensure_collection(qdrant_client)
- index_paths(qdrant_client, genai_client, PATHS)
- Print summary: "Qdrant indexing complete."
- Get collection info and print point count for verification

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Script → Qdrant Cloud | API key sent over HTTPS to cloud endpoint |
| Script → mem0 localhost | HTTP (no TLS) to local mem0 REST API |
| Script → Gemini API | API key sent over HTTPS to Google |
| .env file → shell env | Secrets loaded into process environment |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Information Disclosure | qdrant-indexer.py | mitigate | QDRANT_API_KEY read from os.environ, never hardcoded in script. Verify: `grep -v "environ" qdrant-indexer.py \| grep -c "QDRANT_API_KEY"` returns 0 (no hardcoded key) |
| T-02-02 | Information Disclosure | qdrant-indexer.py | mitigate | GEMINI_API_KEY read via genai.Client() auto-env, never hardcoded. Verify: `grep -c "GEMINI_API_KEY" qdrant-indexer.py` returns 0 (SDK reads it automatically) |
| T-02-03 | Tampering | qdrant-indexer.py | mitigate | assert COLLECTION != "agent_memory" guard prevents accidental writes to mem0's collection |
| T-02-04 | Denial of Service | qdrant-indexer.py | mitigate | time.sleep(0.1) between embed calls + retry with backoff prevents Gemini rate limit exhaustion |
| T-02-05 | Information Disclosure | mem0-export.sh | accept | mem0 accessed over localhost HTTP — acceptable for local-only service, no PII leaves machine |
| T-02-06 | Spoofing | mem0-export.sh | accept | mem0 has no auth — local service, no external exposure. Input is read-only (GET only). |
