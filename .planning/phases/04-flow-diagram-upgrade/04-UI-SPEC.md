## Spacing Scale

All values are multiples of 4. The existing codebase uses Tailwind utility classes mapping to these tokens.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding (`gap-1`, `px-1`, `py-0.5`) |
| sm | 8px | Compact element spacing (`gap-2`, `p-2`) |
| md | 16px | Default panel padding (`p-4`) |
| lg | 24px | Section gap (`gap-6`) |
| xl | 32px | Layout gaps |
| 2xl | 48px | Major section breaks |
| 3xl | 64px | Page-level spacing |

**Exceptions:**
- FlowNode container: fixed 90px width (hardcoded, not spacing scale — retain per D-02)
- FlowNode icon box: fixed 80x80px (retain per D-02)
- Node detail panel: fixed 320px (w-80) width (retain — existing pattern)
- Canvas height: 720px (increased from 620px per D-17)
- Icon touch area inside node box: 80px meets minimum accessibility (44px) — no exception needed

## Node Detail Panel: Heartbeat Section

Appears below the stats grid when `HEARTBEAT_STATE.md` is found.

| State | Behavior |
|-------|----------|
| File found | Show section header "Last State", render last 20 lines as `<pre>` in `font-mono text-xs text-slate-300` with `overflow-x-auto` |
| File not found | Show nothing — hide section entirely (graceful degradation, D-05) |
| Loading | Show single line "Loading state..." in `text-xs text-slate-500` |
| Error fetching | Show nothing — treat as file-not-found |

Section label toggle (D-06):
- Default: "Node Activity"
- When heartbeat content available: "Last State"

## Activity Feed: Noise Cleanup Contract

Regex transforms applied in `/api/activity/route.ts` before events are returned. Pure string transforms, no LLM (D-07, D-08).

| Pattern | Transform |
|---------|-----------|
| `===` surrounded text | Strip `===` delimiters, trim whitespace, keep inner text |
| `---` surrounded text | Strip `---` delimiters |
| Leading ISO-8601 timestamp `YYYY-MM-DDTHH:MM:SS` | Remove |
| `[timestamp]` prefix bracket form | Remove bracket + timestamp |
| Trailing ISO-8601 timestamps mid-string | Remove |
| Lone words: "Starting", "Complete", "cycle" | Strip when they appear alone as the entire message |

Example: `=== APO Cycle Starting 2026-04-09T14:33:22 ===` → `APO Cycle Starting`

|-------------|----------|
