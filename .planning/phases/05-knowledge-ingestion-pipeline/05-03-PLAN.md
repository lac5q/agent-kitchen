**Function: `gws_cmd(args: list) -> dict`**
Same as email script: run via subprocess, parse JSON stdout, return dict or `{}` on failure.

**Function: `ingest_calendar(state: dict) -> tuple[dict, list[dict]]`**
1. Calculate time window: `timeMin = state["calendar"]["last_run"]`, `timeMax = now ISO8601`
2. Call: `gws calendar events list --params '{"calendarId": "primary", "timeMin": "{timeMin}", "timeMax": "{timeMax}", "singleEvents": true, "maxResults": 250}'`
3. Parse events from response `items` field
4. Filter out events whose `id` is in `state["calendar"]["ingested_event_ids"]`
5. For each event:
   a. Extract: `id`, `summary` (title), `start.dateTime`, `attendees` (list of emails), `description`, `conferenceData.conferenceId`
   b. Format attendee list: comma-separated emails
   c. Build mem0 text: `"Calendar event: {summary} on {date} with {attendees}. Description: {description or 'none'}"`
   d. Call `add_mem0(text, "luis", {"type": "calendar_event", "event_id": id, "date": date_str})`
   e. Append event_id to state
   f. Collect event dict for later matching: `{"id": id, "title": summary, "date": date_str, "attendees": attendees, "conference_id": conf_id}`
6. Return updated state and list of processed events (for Drive doc matching)

**Function: `ingest_drive_transcripts(state: dict, calendar_events: list[dict]) -> dict`**
1. List Drive docs: `gws drive files list --params '{"q": "name contains 'Notes by Gemini' and modifiedTime > '{state["gdrive_meet"]["last_run"]}'", "pageSize": 100, "fields": "files(id,name,createdTime,modifiedTime)"}'`
2. Parse `files` from response
3. Filter out docs whose `id` is in `state["gdrive_meet"]["ingested_doc_ids"]`
4. For each doc:
   a. Extract date from name using regex: `re.search(r'(\d{4})/(\d{2})/(\d{2})', doc["name"])` → `doc_date = "YYYY-MM-DD"`
   b. Extract title from name: everything before the date pattern (strip trailing ` - `)
   c. Export content: `gws drive files export --params '{"fileId": "{doc_id}", "mimeType": "text/plain"}' --output /tmp/transcript-{doc_id}.txt`
   d. Read exported file content, then delete temp file
   e. **Multi-event collision check (D-13):** Find all calendar_events matching this doc's date. Match by: same date AND (title substring match OR same conferenceId). If multiple matches → shared_doc=true.
   f. Save canonical transcript once: `MEET_DIR / "{doc_date}-{doc_id[:15]}.md"` with frontmatter:
      ```
      ---
      doc_id: {doc_id}
      title: {title}
      date: {doc_date}
      source: google_drive
      ---
      
      # {title}
      
      {transcript_content}
      ```
   g. **If single event match:** Create project meeting note (normal case)
   h. **If multiple event matches (D-14):** For each matching event, create an index note at `PROJECTS_DIR / {project_name} / meetings / {date}-{slug}-index.md`:
      ```
      ---
      date: {event_date}
      event_id: {event_id}
      attendees: [{attendee_emails}]
      source_doc_id: {doc_id}
      shared_doc: true
      ---
      
      # Meeting -- {event_date}
      
      {first 500 chars of transcript as excerpt}
      
      Source: knowledge/gdrive/meet-recordings/{doc_date}-{doc_id[:15]}.md
      ```
   i. **Project name inference:** Extract from event title — look for known project keywords or attendee patterns. Simple heuristic: if title contains a recognizable project name (e.g., company name from attendees domain), use that. Otherwise fall back to `general`.
   j. Add mem0 memory: `add_mem0("Meeting transcript: {title} on {date} with {attendees}. Key topics: {first_200_chars}", "luis", {"type": "meeting_transcript", "doc_id": doc_id, "date": doc_date})`
   k. Append doc_id to state
   l. `time.sleep(0.5)` between exports (Drive API rate limit)
5. Return updated state

**Function: `build_meetings_section(calendar_events: list[dict], transcript_titles: list[str]) -> str`**
For each calendar event processed, format:
```
### {title} ({date})
Attendees: {comma-separated attendees}
{2-3 sentence summary if transcript available, otherwise "No transcript available."}
```
Join all with `\n\n`.

**Function: `create_project_meeting_note(event: dict, transcript: str) -> None`**
1. Infer project name from event title/attendees (simple heuristic: domain-based or keyword)
2. `project_dir = PROJECTS_DIR / project_name / "meetings"`
3. `project_dir.mkdir(parents=True, exist_ok=True)`
4. Filename: `{date}-{slugify(title)}.md`
5. Write markdown with frontmatter (date, attendees, event_id) + transcript excerpt (first 2000 chars) + link to full canonical file

**Function: `ingest_transcripts() -> None`** (main entry point)
1. `ensure_dirs()`
2. `state = load_state()`
3. `state, events = ingest_calendar(state)`
4. `state = ingest_drive_transcripts(state, events)`
5. Build meetings section content from events
6. If any events processed: `append_daily_note(today, "Meetings", meetings_content)`
7. Update `state["gdrive_meet"]["last_run"]` and `state["calendar"]["last_run"]` to now
8. `save_state(state)`
9. Print summary

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| gws CLI → Calendar API | OAuth-authenticated, reads user's own calendar |
| gws CLI → Drive API | OAuth-authenticated, exports user's own documents |
| Transcript content → markdown file | External content from meeting participants |
| Transcript content → mem0 | Meeting content stored as agent memory |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-09 | I (Info Disclosure) | Meeting transcripts on disk | accept | Same security as any local file; knowledge repo is gitignored; user's own meetings |
| T-05-10 | T (Tampering) | Transcript content in markdown | accept | Content is read-only export from Google Drive; no execution context in .md files |
| T-05-11 | D (DoS) | Unbounded Drive export | mitigate | pageSize=100 cap; 500ms sleep between exports; temp files cleaned after read |
| T-05-12 | T (Tampering) | Path traversal in project name | mitigate | slugify() applied to project name before use in path; strips all non-alnum chars |
| T-05-13 | I (Info Disclosure) | mem0 memories with meeting content | accept | mem0 is local (localhost:3201), same trust as reading your own calendar |
</threat_model>
