2. Create `~/github/knowledge/ingestion_utils.py` with these functions:

- `load_state() -> dict`: Read STATE_PATH (`~/github/knowledge/ingestion-state.json`). If file missing, return DEFAULT_STATE copy. Uses `json.loads(Path.read_text())`.
- `save_state(state: dict) -> None`: Atomic write via write to temp file then `os.replace()` to STATE_PATH. Uses `json.dumps(state, indent=2)`.
- `add_mem0(text: str, agent_id: str = "luis", metadata: dict = None) -> bool`: POST to `http://localhost:3201/memory/add` with `{"text": text, "agent_id": agent_id, "metadata": metadata or {}}`. Returns True on success (HTTP 200), False + prints warning on failure. Uses `requests.post` with 10s timeout.
- `get_body(payload: dict) -> str`: Recursive Gmail body extractor. Handles multipart/alternative — prefers text/plain, falls back to HTML stripped of tags. Decodes base64url body data via `base64.urlsafe_b64decode(data + "==").decode("utf-8", errors="replace")`. Uses `re.sub(r'<[^>]+>', ' ', text)` + `html.unescape()` for HTML stripping.
- `slugify(title: str) -> str`: `re.sub(r'[^a-z0-9]+', '-', title.lower()).strip('-')[:50]`
- `append_daily_note(date_str: str, section: str, content: str) -> None`: Target file is `~/github/knowledge/journals/{date_str}.md`. Create file with `# {date_str}` header if missing. Append `\n\n## {section}\n\n{content}` to end. Uses `>>` semantics (open mode 'a').
- `ensure_dirs() -> None`: `mkdir -p` equivalent for `~/github/knowledge/emails`, `~/github/knowledge/gdrive/meet-recordings`, `~/github/knowledge/spark-recordings`, `~/github/knowledge/projects`, `~/github/knowledge/journals`.
